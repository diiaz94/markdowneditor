# Markdown Editor
 Un editor markdown en tiempo real.
 

## Primer paso

Clonar el repositorio utilizando

### `git clone https://github.com/diiaz94/markdowneditor.git` 

## Segundo paso

Ir a la carpeta clonada y hacer

### `npm install` 

## Tercer paso

Correr el proyecto

### `npm start` 

## Cuarto paso

Ir a la url http://localhost:3000/
